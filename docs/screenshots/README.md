# Screenshot fixtures

The documentation screenshots use fictional connection details and camera data. Do not capture a real Protect URL,
local username, camera name, output path, or camera feed for repository documentation.

`assets/demo-front-door-thumbnail.png` is the fictional camera frame used in the thumbnail popovers. It was generated
for this project as a neutral daytime security-camera view with no people, addresses, license plates, logos, or other
identifying information.
